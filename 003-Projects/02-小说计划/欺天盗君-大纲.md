# 欺天盗君 - 小说大纲

## 📚 作品信息

**小说类型**：奇幻修仙 · 克苏鲁元素 · 智斗复仇

**主角**：吴道（市井小民 → 偷道至尊）

**核心设定**：
- 世界背景：武道昌盛的修仙世界，道魔相争
- 金手指：【偷道空间】（交易身份、寿元、记忆、功法）
- 修炼体系：练气 → 筑基 → 金丹 → 元婴 → 化神 → 道尊 → 羽化

## 🎯 主线剧情

### 第一卷：市井崛起（第1-50章）

#### 第1-5章：初得偷道

- **引子**：吴道在迎春楼后的小巷卖假药，偶遇赌鬼
- **克苏鲁圣像**：赌鬼拿出章鱼头大翅膀的克苏鲁圣像换钱
- **偷道空间**：触摸圣像进入黑雾空间，发现交易能力
- **第一次交易**：与将死老头交易，获得黄金与寿元
- **第一次盗身份**：从赌狗身上偷得青剑山庄四公子身份

#### 第6-15章：山庄风云

- **青剑山庄**：庄主练功走火入魔，夫人掌权
- **高侍卫威胁**：被高侍卫怀疑身份，被迫服毒
- **解毒丹**：通过偷道空间获取解毒丹方
- **毒丹线索**：发现高侍卫是五毒谷间谍
- **家族内部**：大公子与二公子的夺权斗争

#### 第16-30章：复仇与伪装

- **家族秘宝**：得知青剑山庄有先天道气的消息
- **意外刺杀**：魔宗中人刺杀庄主夫人
- **外门试炼**：参加宗门外门弟子选拔
- **功法寻找**：通过偷道空间获取练气功法
- **外门任务**：完成宗门任务，提升实力

### 第二卷：魔道争锋（第51-100章）

#### 第51-65章：魔宗入门

- **魔道功法**：通过偷道空间获取魔道修炼法
- **外门弟子**：加入魔宗成为外门弟子
- **师兄算计**：遭遇师兄妒忌与陷害
- **反向还击**：设计报复，成为外门新秀
- **第二次交易**：获取内门弟子修炼法

#### 第66-85章：内门风云

- **内门晋升**：成功晋升内门弟子
- **内门弟子**：结识重要人物，建立势力
- **正道弟子设计**：袭杀小魔宗获取秘宝
- **盗取正道身份**：盗取正道弟子身份混入正派
- **返回途中**：遭遇意外，揭示更大秘密

### 第三卷：偷道至尊（第101-200章）

#### 第101-130章：偷道真谛

- **五神通**：进入金丹境，获得五种神通
- **道果本质**：道痕凝结成果实，决定大道
- **三界大道**：天道、地道、人道的本质
- **偷道天然**：域外天魔身份的真相
- **道尊陨落**：道尊被联合击杀，神魂传承

#### 第131-170章：逆天之途

- **风道修炼**：参悟风之道，速度达到极致
- **火道真谛**：掌控火焰法则，焚尽万物
- **剑道无双**：剑意通神，一剑破万法
- **无情道心**：斩断情欲，追求大道
- **神通融合**：五种神通融合，威力无穷

#### 第171-200章：欺天盗君

- **羽化境界**：突破化神，进入羽化境界
- **克苏鲁天尊**：前偷道魔尊的真实身份
- **欺世万法**：融合各种大道，成为万法天尊
- **最终对决**：与天道相争，盗取天道权柄
- **结局**：成为偷道至尊，改写天地法则

## 🎭 核心设定

### 🔮 修炼体系

| 境界 | 特征 | 对应神通 |
|------|------|----------|
| 练气 | 吸收天地灵气，强化身体 | 感知天地 |
| 筑基 | 建立道基，沟通天地 | 本命法宝 |
| 金丹 | 大道凝结成丹 | 五神通初现 |
| 元婴 | 灵魂元婴，可分神 | 元婴出窍 |
| 化神 | 元神寄托虚空 | 移山倒海 |
| 道尊 | 掌握一道，开宗立派 | 掌控法则 |
| 羽化 | 超凡入圣，突破天道 | 欺天盗法 |

### 💎 偷道空间

**功能**：
- **身份盗取**：交换他人身份，继承记忆
- **寿元交易**：购买或出卖寿命
- **记忆买卖**：读取或传递记忆
- **功法获取**：购买或窃取修炼法
- **道果融合**：融合各种大道痕迹

**限制**：
- 每一次交易消耗自身寿命
- 境界限制交易范围
- 高等级道果有反噬风险
- 无法直接盗取大道权柄

### 📜 重要人物

#### 主角阵营

- **吴道**：主角，市井小民出身，拥有偷道空间
- **青剑山庄四公子**：吴道盗取的第一个身份
- **高侍卫**：五毒谷间谍，主角敌人
- **庄主夫人**：青剑山庄实际掌权者
- **大公子/二公子**：山庄内部竞争者

#### 反派角色

- **天道化身**：代表天地法则的存在
- **正道修士**：维护秩序的修仙者
- **魔宗长老**：修炼魔道的强大存在
- **域外天魔**：来自异空间的邪恶势力

## 🎨 特色设定

### 克苏鲁元素

- **克苏鲁圣像**：前偷道魔尊欺世万法天尊的道器
- **精神污染**：频繁使用偷道空间会受到精神影响
- **不可名状**：偷道空间内部存在不可名状的恐怖
- **宇宙真相**：终极秘密与克苏鲁神话有关联

### 智斗设计

- **伪装身份**：通过盗取他人身份进行间谍活动
- **信息战**：利用记忆交易获取情报
- **功法窃取**：通过交易获得他人修炼经验
- **道果争夺**：收集各种大道痕迹，融合成完美道果

## 🔮 创作要点

### 主题深化

- **身份认同**：通过身份盗窃引发对存在意义的思考
- **因果循环**：每一次偷道都有对应的因果报应
- **大道之争**：个人欲望与天地法则的对抗
- **终极秘密**：宇宙真相与个人命运的关联

### 节奏控制

- **开篇紧凑**：10章内建立核心冲突
- **中期展开**：引入新势力与设定
- **高潮迭起**：每卷结束有大事件
- **结局圆满**：收尾呼应，解决所有伏笔

- **市井气息**：初期描写真实的市井生活

- **智斗悬疑**：通过逻辑推理推动情节
